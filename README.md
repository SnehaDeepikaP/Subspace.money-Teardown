The teardown was built as part of the **Product Intern Assignment** for VocalLabs.ai / Subspace.money. It covers 5 sharp, non-generic feedbacks across all five product pillars — each grounded in direct app and website usage, real Play Store reviews, and competitor benchmarking.

## What's inside

### 5 Feedbacks across all product pillars

| # | Pillar | Feedback |
|---|--------|----------|
| 1 | **UX** | Homepage gates value behind login before showing a single reason to sign up |
| 2 | **Features / Services** | Shared subscription flow has a trust gap — subscribers pay, admins ghost, no structural protection |
| 3 | **GTM & ICPs** | Messaging tries to be everything to everyone and lands for nobody in particular |
| 4 | **Competitor Analysis** | The Negotiate API (Subspace's biggest moat) is completely invisible to users |
| 5 | **Potential Collaborations** | Partnership surface is idle while ONDC and VocalLabs create a once-in-a-cycle opening |

Each feedback is structured as:

- **(a) Observed** — specific screen, copy, or flow seen in the actual app/website
- **(b) Problem** — the real user or business pain it causes
- **(c) Ship instead** — a concrete, actionable solution

### Supporting analysis

- **SWOT** — Strengths, Weaknesses, Opportunities, Threats
- **Porter's Five Forces** — competitive intensity across all five forces
- **Competitor feature map** — Subspace vs. myPaisaa, MoneyClub, Splitwise India, PhonePe (9 features)
- **Priority matrix** — impact × effort ranking across all 5 feedbacks

## My approach

1. **Used the product like a real customer** — opened subspace.money in an incognito browser window to capture the cold-user experience, then downloaded and walked through the full Android app (WhatsApp onboarding → Shared Subscriptions → Gift Cards → Rentals → Explore tabs)
2. **Read real user feedback** — studied 1-star and 2-star Play Store reviews (April–May 2026); the trust gap (Feedback #2) came directly from a live top-rated negative review
3. **Benchmarked competitors** — mapped Subspace's feature set against myPaisaa, MoneyClub (TMC), Splitwise India, and PhonePe using Tracxn, Play Store listings, and direct product exploration
4. **Applied frameworks** — SWOT and Porter's Five Forces for strategic context; 7Ps lens informally across each feedback
5. **Structured every feedback as Observed → Problem → Ship** — no feedback is generic; each anchors to a specific screen or flow

## Key findings at a glance

| Finding | Impact |
|---------|--------|
| Login-first homepage kills cold-traffic conversion | High — pure layout fix, 1-week ship |
| No subscription escrow = #1 churn trigger in reviews | **Critical** — 2–4 week fix |
| Negotiate API is invisible = moat going to waste | High — surfacing it costs nothing |
| Fragmented ICP messaging inflates CAC | High — messaging fix + referral programme |
| VocalLabs × ONDC integration sitting unused | High (strategic) — founder owns both products |
